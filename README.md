Food Serving Multi-Robotic System

Overview

This project is a Java-based simulation of a food-serving robotic system designed for a large restaurant environment. The application models how autonomous service robots can safely operate in crowded dining areas while respecting occupancy limits and maintaining physical distancing requirements.

The system evaluates whether robots can enter specific restaurant locations based on capacity constraints and continuously monitors the distance between robots and nearby people to ensure safe operation.

Features

* Restaurant occupancy management
* Capacity-based access control
* Robot movement simulation
* Social distancing monitoring
* Dynamic distance calculations
* Contact status classification
* Safety recommendations for robot movement
* Object-oriented system design
* User-driven simulation through console input

Example Scenario

The system determines whether a robot can enter a restaurant zone based on current occupancy levels. Once inside, distances between robots and nearby individuals are evaluated. If distancing requirements are violated, the application generates recommendations and classifies the interaction status accordingly.
